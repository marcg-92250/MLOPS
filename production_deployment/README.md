# ML Model Deployment Project

Deploy a web service running a machine learning model with Docker.

## Project Structure

```
production_deployment/
├── level0_local/          # Level 0: Local machine deployment
├── level0.5_remote/       # Level 0.5: Remote machine deployment
├── level1_docker/         # Level 1: Docker containerization
├── level2_cloud/          # Level 2: Cloud VM deployment
├── level3_cicd/           # Level 3: CI/CD automation
├── data/                  # Dataset
└── models/                # Trained models
```

## Prerequisites

```bash
pip install fastapi uvicorn scikit-learn pandas joblib requests
```
